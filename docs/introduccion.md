# Introducción


El proyecto "Generador de contraseñas seguras" nace de la necesidad de facilitar a usuarios particulares, profesionales y empresas la creación de credenciales robustas que protejan la información frente a ataques cibernéticos. Una contraseña segura debe combinar distintos tipos de caracteres y tener longitud adecuada para evitar ataques de fuerza bruta o ataques por diccionario.


Este documento acompaña al código fuente y explica las decisiones de diseño, uso y funcionalidades implementadas.

# Generador de contraseñas seguras


**Generador de contraseñas seguras** es una herramienta de software que crea contraseñas robustas combinando mayúsculas, minúsculas, números y símbolos. Permite al usuario especificar la longitud de la contraseña, garantiza la inclusión de los tipos de caracteres necesarios y puede guardar las contraseñas generadas en un archivo para su gestión posterior.


## Características
- Generación de contraseñas personalizables por longitud.
- Asegura al menos un carácter de cada tipo: mayúscula, minúscula, número y símbolo (cuando la longitud lo permite).
- Opción para guardar las contraseñas generadas en un archivo local.
- Interfaz por línea de comandos (fácil de usar y adaptar).


## Estructura del repositorio
- `codigo/` : Código fuente del generador.
- `docs/` : Documentación (Introducción, Estado del Arte, Manual de usuario, Conclusiones).
- `img/` : Imágenes y diagramas del proyecto.
- `pruebas/` : Ejemplos y archivos de prueba.
